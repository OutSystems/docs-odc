# OS-AICA-50001

## Error message

```
There was an unknown error.
```

## Cause

An unexpected error occurred while Mentor Web was processing the request.

## Impact

Mentor Web can't complete the current request.

## Recommended action

Refresh the page and try again with a different prompt, since specific inputs can trigger this failure. If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-AICA-50001).
