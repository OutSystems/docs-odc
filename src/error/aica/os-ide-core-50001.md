# OS-IDE-CORE-50001

## Error message

```
Sorry, we can't handle your request.
```

## Cause

An unexpected error occurred elsewhere in the platform. OS-IDE-CORE-50001 is a generic code that wraps failures that don't have a more specific error code.

## Impact

Mentor Web can't complete the current operation, such as loading an app, applying a change, or publishing.

## Recommended action

Retry the action. If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-IDE-CORE-50001), including the trace ID shown in the error details.
