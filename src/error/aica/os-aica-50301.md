# OS-AICA-50301

## Error message

```
The LLM Gateway is currently down.
```

## Cause

Mentor Web can't reach the LLM Gateway due to an outage, a connectivity issue, or a maintenance window.

## Impact

Mentor Web can't process any request until the gateway recovers.

## Recommended action

Wait a few minutes and try again. If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-AICA-50301).
