# OS-CTXS-50202

## Error message

```
Could not authenticate with or is not authorized to access an upstream service.
```

## Cause

The Context Service failed to authenticate with one of its upstream dependencies, or it authenticated but wasn't authorized for the operation.

## Impact

The request didn't complete.

## Recommended action

Create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-CTXS-50202) and include the error code.
