# OS-CTXS-50001

## Error message

```
Server encountered an error and was unable to respond.
```

## Cause

The Context Service hit an unexpected internal error while processing the request.

## Impact

The request didn't complete.

## Recommended action

Create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-CTXS-50001) and include the error code.
