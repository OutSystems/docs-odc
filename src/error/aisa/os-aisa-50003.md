# OS-AISA-50003

## Error message

```
Problem initializing connection.
```

## Cause

Mentor couldn't establish a connection to the AI service.

## Impact

Mentor can't process requests.

## Recommended action

Try again. If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-AISA-50003).
