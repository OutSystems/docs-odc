# OS-AISA-40002

## Error message

```
Mentor ran into an issue. Please clear the chat and try again.
```

## Cause

An unsupported change occurred in the conversation's processing mode between turns.

## Impact

Mentor can't continue the current conversation.

## Recommended action

Clear the chat and start a new conversation.
