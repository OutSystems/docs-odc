# OS-AISA-42903

## Error message

```
Mentor is currently unavailable due to reached usage limits. Limit resets in <reset_time>.
```

## Cause

The organization reached its daily usage limit for Mentor.

## Impact

Mentor is unavailable until the limit resets at the time shown in the error message.

## Recommended action

Wait for the limit to reset at the time indicated in the error message.
