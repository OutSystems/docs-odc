# OS-AISA-50301

## Error message

```
LLM Gateway is down.
```

## Cause

The AI Provider service that powers Mentor is temporarily unavailable.

## Impact

Mentor can't process any requests until the service is restored.

## Recommended action

Wait a few minutes and try again. If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-AISA-50301).
