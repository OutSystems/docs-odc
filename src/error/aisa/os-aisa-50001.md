# OS-AISA-50001

## Error message

```
Mentor ran into an issue. Please clear the chat and try again.
```

## Cause

An unexpected error occurred while Mentor was processing the request.

## Impact

Mentor can't complete the current request.

## Recommended action

Clear the chat and try again with a different prompt, since specific inputs can trigger this failure. If the problem persists, create a case with [OutSystems Support](https://www.outsystems.com/support/portal/open-support-case?ErrorCode=OS-AISA-50001).
